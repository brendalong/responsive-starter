Bootstrap 4: Grid Example

Bootstrap's visible and hide classes. Examine the following grids in a browser. This first grid is using all grid sizes combined in a single row. Start by adding the d-none class. Work up from the smallest to display content at different sizes.

xl indicates that the x-large grid is displaying. The grid stacks horizontally < 1200px.

lg indicates that the large grid is displaying. The grid stacks horizontally < 992px.

md indicates that the medium grid is displaying. The grid stacks horizontally < 768px.

sm indicates that the small grid is displaying. The grid stacks horizontally < 575px.

xs indicates that the extra small grid is displaying. This grid stays horizontal.
